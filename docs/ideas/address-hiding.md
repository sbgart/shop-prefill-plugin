# Идея: Скрытие адресов из списка выбора

## Описание

Дать возможность пользователю (авторизованному или гостю) скрывать некоторые адреса из списка выбора "Мои адреса".

## Реализация

### Подход
Добавить параметр `prefill_address_hidden = 1` в `shop_order_params` для заказов, которые не должны использоваться для набора адресов.

### Преимущества
- Работает для авторизованных (по `contact_id`) и гостей (по `guest_hash`)
- Не требует новых таблиц
- Легко фильтровать при выборке заказов

### UI
- Кнопка "Скрыть" рядом с каждым адресом в диалоге выбора
- AJAX-запрос для установки `prefill_address_hidden = 1`

### Связанные задачи
- Диалог выбора адресов (уже есть базовая реализация)
- Consent-механизм (реализуется сейчас)
